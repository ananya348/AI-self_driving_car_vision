## 🔥 Features

✅ 🚘 Real-time Object Detection (YOLOv8)
✅ 📏 Distance Estimation (Monocular Vision)
✅ 🧠 Scene Reasoning (AI Decision Making)
✅ ⚠️ Collision Risk Prediction (LOW / MEDIUM / HIGH)
✅ 📊 Tesla-style Dashboard UI
✅ 🚗 Object Counter (Cars, People, Trucks, Buses)
✅ 🎥 Video-based Detection System

---

## 🧠 How It Works

1. 🎥 Upload a driving video
2. 🔍 Detect objects (cars, pedestrians, etc.)
3. 📏 Estimate distance using bounding box scaling
4. 🧠 Analyze scene for potential risks
5. 🚨 Display alerts & risk levels in real-time
